4:77:4448:23:19:72:542:689:14:Times:14:Times:C:The regional effects of both national and sub-national taxes have been a topic of policy

14:411:25350:24:20:54:286:350:15:Times:15:Times:C:Implementing the Modelling Approach
14:412:25385:20:19:79:521:330:13:Times:13:Times:A:To apply the model outlined above to the evaluation of the interregional effects of taxes
14:413:25474:19:20:55:520:311:13:Times:13:Times:B:in Canada, parameter values must be specified for the functions used in the model, and it
14:414:25563:20:21:55:503:291:13:Times:13:Times:B:must be solved for competitive equilibria under the various policy changes considered.
14:415:25649:21:20:79:518:270:13:Times:13:Times:A:In specifying model parameter values, a calibration procedure similar to those used
14:416:25732:20:18:55:519:250:13:Times:13:Times:B:in other applied general equilibrium models is followed (see Mansur and Whalley (1984)).
14:417:25820:18:19:55:519:232:14:Times:14:Times:C:Calibration is most easily understood as the use of model equilibrium conditions and
14:418:25904:19:19:55:520:213:14:Times:14:Times:C:equilibrium data to solve for the parameter values used in the functions in the model.
14:419:25990:19:20:56:521:194:14:Times:14:Times:C:This involves selecting a set of parameters values such that data which characterizes a
14:420:26077:20:19:56:518:174:13:Times:13:Times:B:benchmark (or observed) equilibrium can be reproduced as a model equilibrium. Only
14:421:26159:19:18:56:519:155:13:Times:13:Times:B:when the model is fully specified and a policy change incorporated is the model solved
14:422:26245:18:20:56:518:137:13:Times:13:Times:B:for a new equilibrium solution. Evaluations of the regional impacts of taxes follow from
14:423:26333:20:18:56:521:117:13:Times:13:Times:B:pairwise comparisons between simulated (or new) equilibria and the benchmark equilibrium
14:424:26421:18:21:56:227:99:13:Times:13:Times:B:to which the model is calibrated.
14:425:26454:21:29:80:520:78:13:Times:13:Times:A:Thus, two types of equilibria have to be distinguished when using the model. One is
14:426:26537:29:0:286:291:49:10:Times:10:Times:F:9

15:428:26625:18:19:80:545:685:13:Times:13:Times:A:calibrated (and thus do not need to be computed). The second is 'new' or 'counterfactual'
15:429:26714:19:20:80:463:666:13:Times:13:Times:A:equilibria which are computed as model solutions under changes in policy.
15:430:26787:20:19:104:543:646:13:Times:13:Times:A:Elasticity estimates enter this calibration process by serving as identifying restrictions,
15:431:26878:19:19:79:541:627:13:Times:13:Times:A:allowing the other parameter values in the model to be directly calculated. Since different
15:432:26969:19:19:79:541:608:13:Times:12:Times:A:elasticities produce changed values for the other model parameters, therebyaffecting model
15:433:27059:19:18:79:541:589:13:Times:13:Times:A:results, selecting appropriate elasticity values is central to the model specificationprocess.
